Brak połączenia z bazą

Aplikacja nie ma poprawnej konfiguracji Supabase albo zmienne nie zostały wczytane po deployu.

{{ message }}

Na Vercelu: Project → Settings → Environment Variables — dodaj dokładnie:

  • SUPABASE_URL (Project URL z Supabase)
  • SUPABASE_SERVICE_ROLE_KEY (klucz service_role, nie anon)

Zapisz zmienne, potem Deployments → ostatni deploy → ⋯ → Redeploy.
